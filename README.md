# 🧮 JavaScript Calculator

A simple browser-based calculator built with HTML, CSS, and vanilla JavaScript. Developed as a mini project during the [SuperSimpleDev](https://www.youtube.com/@SuperSimpleDev) JavaScript course.

## 🔗 Live Demo

👉 [View it live on GitHub Pages](https://ah-07.github.io/Calculator/)

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Decimal number support
- Persistent state via `localStorage` — your calculation is saved even if you refresh the page
- Clear button to reset the display
- Styled with a dark theme inspired by iOS Calculator

## How to Use

Just open the live demo link above — no installation needed!

Or to run it locally:
1. Clone the repository
2. Open `index.html` in your browser

## Project Structure
```
├── index.html             # Markup and button layout
├── CalculatorFinal.css    # Dark theme styling
└── CalculatorFinal.js     # Calculation logic and localStorage
```

## What I Learned

- Handling user input with `onclick` event handlers
- Using `eval()` to compute mathematical expressions
- Persisting data across page reloads with `localStorage`
- Styling buttons with CSS (border-radius, colors, sizing)
